    • no -> Dump Memory to File

  • Common Win32 APIs to look out for
    • VirtualAlloc and all of its variants
    • VirtualProtect
      • Modifies memory page permissions
      • Some malware may allocate a new memory with R/W but later change it to RWX with VirtualProtect
    • RtlDecompressBuffer
      • Decompresses buffer that is compressed using COMPRESSION_FORMAT_LZNT1
    • LoadLibrary
      • Loads additional libraries (DLLs)
      • e.g., LoadLibrary("advapi32.dll")
    • GetProcAddress
      • Gets additional exports from libraries
      • e.g., GetProcAddress(hAdvapi32, "RegSetValueEx")
